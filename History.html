History

2024.01

Paper published 📖

You can download the paper DIFFUSUP: A graphical user interface (GUI) software for diffusion modeling from this link.

2022.01

Website 💻

DIFFUSUP's website was built!

2021.11

Huge update V1.10 🍎

The whole software including GUI, algorithm and functions were all updated.

2021.07

Huge update V1.00 🌲

The GUI design changed a lot. This is the first version that considered diffusion in different mediums.

2021.04

Original version 🍀

The first version of DIFFUSUP was construted. The first version containes GUI but lack a lot of functions. The first version can only conduct divalent cations diffusion modeling in garnet and carbonate.

2020.06

Prototype of diffusup 🌱

The authors built the prototype of DIFFUSUP and tested in their own projects.

2019.10

First version function 🔢

The authors wrote the first version MATLAB '.m' file that contained the Crank-Nicolson algorithm. This '.m' file further was further modified and used in DIFFUSUP.
