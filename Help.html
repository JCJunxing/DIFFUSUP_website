FAQ

Is DIFFUSUP free to use?

Yes. You can download, use DIFFUSUP in your project for free.

Is DIFFUSUP safe to use?

Yes. DIFFUSUP is safe to use. The whole code set can be download here.

How to cite if you are using DIFFUSUP?

Currently, You can download the paper DIFFUSUP: A graphical user interface (GUI) software for diffusion modeling from this link.

What is needed for a calculation in DIFFUSUP?

For the calculation in DIFFUSUP, the inital profile, diffusion condition (eg.temperature, pressure, diffusion rate) and diffusion time is needed. The output is the numerical modeling of the diffused profile.
